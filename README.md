# ◉ Orion GUI

A sleek, modern GUI for **Orion AI** running on **Ollama**.

Created by **OmniNode**

## Features

- 🎨 Modern dark UI with glow effects
- 💬 Real-time streaming responses
- 🔄 Model switching
- 📱 Mobile responsive
- 📜 Chat history
- ⚡ Fast and lightweight

## Requirements

- Python 3.8+
- Ollama running locally
- Orion model installed

## Setup

```bash
git clone https://github.com/OmniNode/Orion.git
cd Orion
pip install -r requirements.txt
python app.py
